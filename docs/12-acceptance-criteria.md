# Acceptance Criteria

## Login

### Given
The user is registered.

### When
The user enters valid credentials.

### Then
The system grants access to the dashboard.

---

## Attendance

### Given
The employee is logged in.

### When
The employee clicks "Mark Attendance".

### Then
Attendance is successfully recorded.

---

## Leave Application

### Given
The employee is logged in.

### When
The employee submits a leave request.

### Then
The request is sent to HR for approval.

---

## Leave Approval

### Given
HR Manager is logged in.

### When
HR approves a leave request.

### Then
The employee receives an approval notification.

---

## Report Generation

### Given
Admin selects report options.

### When
Generate Report is clicked.

### Then
The report is displayed successfully.
