# Functional Requirements

## Project Name

Employee Leave & Attendance Management System

---

# Introduction

This document describes the functional requirements of the Employee Leave & Attendance Management System. Functional requirements define the features and services that the system must provide to its users.

---

# User Authentication

## FR-01 User Login

Description:
The system shall allow registered users to log in using their username and password.

Priority:
High

---

## FR-02 User Logout

Description:
The system shall allow authenticated users to securely log out.

Priority:
High

---

# Employee Management

## FR-03 Add Employee

Description:
Administrator shall add new employee information.

Priority:
High

---

## FR-04 Update Employee

Description:
Administrator shall edit employee information.

Priority:
Medium

---

## FR-05 Delete Employee

Description:
Administrator shall remove employee records.

Priority:
Medium

---

## FR-06 Search Employee

Description:
Users shall search employees using ID or name.

Priority:
Medium

---

# Attendance Management

## FR-07 Mark Attendance

Description:

Priority:
High

## FR-08 View Attendance

Priority:
High

---

## FR-09 Attendance Report

Description:

Priority:
High

# Leave Management

Description:

High

## FR-11 View Leave Status

Priority:
High

---

## FR-12 Approve Leave

Description:

Priority:
High

High

Priority:
Medium

## FR-16 Leave Report

## FR-18 Update Profile

The system contains twenty functional requirements covering authentication, attendance, leave management, employee management, reporting, notifications, and security.
