# User Stories

## Employee

- As an Employee, I want to log in securely so that I can access my dashboard.
- As an Employee, I want to mark my attendance so that my work hours are recorded.
- As an Employee, I want to apply for leave online so that I do not need paper forms.
- As an Employee, I want to check my leave status so that I know whether my leave is approved.

---

## HR Manager

- As an HR Manager, I want to review leave requests so that I can approve or reject them.
- As an HR Manager, I want to manage employee records.
- As an HR Manager, I want to generate attendance reports.

---

## Administrator

- As an Administrator, I want to manage users.
- As an Administrator, I want to configure system settings.
- As an Administrator, I want to monitor system activities.
